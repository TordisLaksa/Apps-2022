# React & Ionic

- Opret et projekt
- Arbejde med Ionic komponenter
- Arbejde med Overlays

## Installer Ionic CLI (Command Line Interface)
```
npm install -g @ionic/cli
```

## Opret et projekt

Start Ionic projekt:
```
ionic start
```
- Vælg framework (Angular, React, Vue)
- Angiv projektets navn
- Vælg installationstype (Blank)
